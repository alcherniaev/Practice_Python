#Ќапишите программу, котора€ считывает с клавиатуры два числа a и b,
#считает и выводит на консоль среднее арифметическое всех чисел из отрезка [a;b], 
#которые дел€тс€ на 3.

a,b = [int(input()) for i in range(2)]
sum = 0
n = 0
for i in range(a, b+1):
  if i%3 == 0:
    sum += i
    n += 1
print(sum / n)
